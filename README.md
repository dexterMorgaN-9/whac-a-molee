# Whac-A-Mole

A small Whac-A-Mole game I made in Godot to practice game development.

The goal is simple: click the moles as fast as you can before the timer runs out. Your best score is saved locally, so you always have something to beat.

## Features

- 60-second game rounds
- High score saving
- Random mole spawning
- Simple main menu
- Quick restart

## Controls

| Action | Input |
| ------ | ----- |
| Hit a mole | Left Mouse Button |
