# Whac-A-Mole

A fast-paced arcade game inspired by the classic Whac-A-Mole. Test your reflexes, smash as many moles as possible before time runs out, and beat your highest score.

## Controls

| Action | Input |
| ------ | ----- |
| Hit a mole | Left Mouse Button |
